
* Совместим с версией ядра 1С-Битрикс 17.х(для версии 16.х тоже подойдет)
* Версия модуля 1.2

Обработчик платежной системы для 1С-Битрикс Провайдер Интеркасса https://www.interkassa.com

1. Убедитесь в соответствии версий модуля и вашей CMS, они должны совпадать.
2. Распаковать в корень сайта.  
3. Зайти в админ панели в раздел Магазин->Платежные системы->Добавить новый, выбрать из списка Интеркассу.
4. Заполнить все поля настроек.
5. После создания Платежной системы сайта, ей будет присвоен ID(идентификатор), 
который надо будет ввести в значение переменной 'PAY_SYSTEM_ID_NEW' 
в файле расположеного по пути [корень_сайта]/payment-interkassa/callback.php
