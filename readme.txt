
* Совместим с версией ядра 1С-Битрикс 16.x
* Версия модуля 1.1

Обработчик платежной системы для 1С-Битрикс Провайдер Интеркасса https://www.interkassa.com

1. Распаковать в корень сайта.  
2. Зайти в админ панели в раздел Магазин->Платежные системы->Добавить новый, выбрать из списка Интеркассу.
3. Инструкции по настройке содержаться в административной части сайта.
